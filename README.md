# API Consumer App with Fetch and Axios

Mandatory exercise for Sprint 3. Web application that consumes data from an external API ([JSONPlaceholder](https://jsonplaceholder.typicode.com/)) using both **Fetch API** and the **Axios** library, with search, pagination and UI state management. The user can select the request method (Fetch or Axios), enter a search term and get paginated results displayed as cards. The application handles loading, error and empty results states.

## Technologies

- Frontend: HTML5, CSS3, JavaScript (ES6+)
- API: [JSONPlaceholder](https://jsonplaceholder.typicode.com/)
- HTTP Library: [Axios](https://axios-http.com/) via CDN

## Installation

1. Clone the repository: `git clone https://github.com/bertagonzalezgu/task-S3.01.git`
2. Open `index.html` in your browser

No dependencies or server required.
